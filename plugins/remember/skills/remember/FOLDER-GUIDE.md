# Folder Guide

## Quick Reference

| Folder | Type | When to Use |
|--------|------|-------------|
| `learnings/` | learning | Validated patterns (used 3+ times) |
| `decisions/` | decision | Choices made with rationale |
| `troubleshooting/` | troubleshooting | Problems solved, root causes found |
| `research/` | research | Analysis, comparisons, exploration |
| `backlog/` | note | Future work, ideas, TODOs |
| `context/` | note | Project info, preferences, setup |
| `archive/` | varies | Completed or superseded content |

## Decision Tree

```
Validated pattern used 3+ times?  --> learnings/
Choice made with rationale?       --> decisions/
Problem solved / debugged?        --> troubleshooting/
Analysis or comparison?           --> research/
Future task or idea?              --> backlog/
Otherwise                         --> context/
```

## Tags

Use 3-5 domain tags (e.g., `kubernetes`, `storage`, `home-automation`).

Avoid tags that duplicate the type or status.
